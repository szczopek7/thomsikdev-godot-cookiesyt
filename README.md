# thomsikdev-godot-cookiesyt

ThomsikDev Godot Cookies is a small idle/clicker game prototype built with Godot Engine and GDScript.

This project focuses on learning scalable game architecture, modular UI design, and incremental-game mechanics such as machines, upgrades, and skills.

#More information

[Youtube - Course](https://www.youtube.com/watch?v=B3cRIsV03HA&list=PLFuUnpEI4Goz96EA-U8vlGphCOBQeEqhB&index=1 "Youtube - Course")

#Requirements

Godot Engine 4.6

Installation
`git clone https://github.com/szczopek7/thomsikdev-godot-cookiesyt`

1. Open Godot Project Manager
2. Click Import
3. Select the project folder
4. Run the main scene
